# User Guide

## Navigation

Learn how to navigate through the WikiTest application effectively.

### Main Menu

The main menu provides access to all major features:
- **Home**: Dashboard with recent activity
- **Documents**: Browse and search documents
- **Profile**: Manage your user profile

## Creating Content

### Adding a New Document

1. Click on "Documents" in the main menu
2. Select "Create New"
3. Fill in the required fields
4. Click "Save"

## Searching

Use the search bar at the top of the page to find documents quickly.

### Search Tips

- Use quotes for exact phrases
- Use wildcards (*) for partial matches
- Filter by date, author, or category

## Best Practices

- Keep document titles clear and descriptive
- Use tags to organize content
- Review and update content regularly
